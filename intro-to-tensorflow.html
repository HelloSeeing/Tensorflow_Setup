什么是Tensorflow

​

TensorFlow™ 是一个使用数据流图进行数值计算的开源软件库。图中的节点代表数学运算， 而图中的边则代表在这些节点之间传递的多维数组（张量）。这种灵活的架构可让您使用一个 API 将计算工作部署到桌面设备、服务器或者移动设备中的一个或多个 CPU 或 GPU。 TensorFlow 最初是由 Google 机器智能研究部门的 Google Brain 团队中的研究人员和工程师开发的，用于进行机器学习和深度神经网络研究， 但它是一个非常基础的系统，因此也可以应用于众多其他领域。

这段话摘自Tensorflow中文官网, 其实在课程一开始, 大家可以把它当作一个比较成熟, 非常流行, 功能非常强大的工具. 而这个工具在我们日后的学习中会一直出现, 大家对他的理解也会日渐深刻. 而在这里, 我们把它当成一把锤子.

Tensorflow-1.4.0 安装

Tensorflow 的安装作为我们正是开始学习的最初一步, 具有非常重要的意义. 网络上已经有非常多丰富的教程来指导大家完成这个可能并不复杂的安装. 那么在这里我们也把自己实际操作的经验列在下方作为最后一个部分, 希望大家安装愉快!

环境
  • 系统: Ubuntu
  • python2.7 或者 python3.4+
确定你想要的tf版本

因为每台电脑的硬件配置不同, Tensorflow 具有不同的安装方式. 简单来说, 如果你的电脑拥有具有较高计算能力(Compute Capabilit>=3.0)的GPU, 那么你可以安装GPU版本, 这会让你的程序运转如飞; 反之, 那么你只能选择较低配置的CPU版进行安装, 但没关系, 速度稍微慢一点并不影响我们学习的进度和心情.

如果具有符合条件的GPU, 那么安装的过程会稍显麻烦一点, 需要安装好下面几个依赖项才能够开始Tensorflow的安装(安装CPU版本的话可以跳过这阶段):

CUDA 8.0

CUDA 8.0 作为GPU程序的编译环境是安装过程中比较重要的一步. 首先去这个网址按照你的机器的硬件配置进行下载的选择. 例如我是Ubuntu 14.04, 那么我的选项依次是Linux->x86_64->Ubuntu->14.04->deb(local).之后弹出一个带有下载链接的框以及安装说明, 照着顺序运行命令即可.

按照说明安装完成后, 还需要将cuda添加到系统环境中去, 那么还需要运行以下命令:

 
$ sudo vi ~/.bashrc

在最后添加

 
xxxxxxxxxx
export PATH=/usr/local/cuda-8.0/bin${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-8.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

保存之后运行

 
xxxxxxxxxx
$ source ~/.bashrc
$ nvcc -V

如果出现类似下面的输出则说明安装成功

 
xxxxxxxxxx
nvcc: NVIDIA (R) Cuda compiler driver
Copyright (c) 2005-2016 NVIDIA Corporation
Built on Sun_Sep__4_22:14:01_CDT_2016
Cuda compilation tools, release 8.0, V8.0.44

如果出现错误, 可以参考 nvidia 的指导安装网页进行修正.

Cudnn v0.6

cudnn 作为cuda的加速计算库是安装过程比较简单的一步. 首先从这个网址下载到本地, 然后运行下列命令:

 
xxxxxxxxxx
$ tar -xzvf cudnn-*.*-linux-***-v*.tgz
$ sudo cp cuda/include/cudnn.h /usr/local/cuda/include 
$ sudo cp cuda/lib64/libcudnn* /usr/local/cuda/lib64 
$ sudo chmod a+r /usr/local/cuda/include/cudnn.h /usr/local/cuda/lib64/libcudnn*
Cupti

运行下列命令安装即可

 
xxxxxxxxxx
$ sudo apt-get install libcupti-dev
安装Tensorflow

这些与nvidia有关的依赖项安装完成后, 就可以正式开始安装Tensorflow了. 这里我们提供两种安装方法

  • pip
  • anaconda

pip 安装

  • 首先运行下列命令:
 
xxxxxxxxxx
$ pip install tensorflow      # Python 2.7; CPU support (no GPU support)
$ pip3 install tensorflow     # Python 3.n; CPU support (no GPU support)
$ pip install tensorflow-gpu  # Python 2.7;  GPU support
$ pip3 install tensorflow-gpu # Python 3.n; GPU support
  • 如果上面失败了的话, 可以运行下列命令:
 
xxxxxxxxxx
$ sudo pip  install --upgrade tfBinaryURL   # Python 2.7
$ sudo pip3 install --upgrade tfBinaryURL   # Python 3.n

这里tfBinaryURL是根据你电脑配置不同而不同的链接地址, 如果你是Linux, python3.4, CPU-Only的安装配置的话, 可以这样运行:

 
xxxxxxxxxx
$ sudo pip3 install --upgrade \
 https://storage.googleapis.com/tensorflow/linux/cpu/tensorflow-1.4.0-cp34-cp34m-linux_x86_64.whl
  • 之后可以对你的安装进行验证

anaconda安装

  • 首先安装anaconda

  • 创建一个 tensorflow 的 conda 环境, 例如:

     
    xxxxxxxxxx
    $ conda create -n tensorflow python=2.7 # or python=3.3, etc.

  • 激活 conda 环境

     
    xxxxxxxxxx
    $ source activate tensorflow
    (tensorflow)$

  • 然后运行下列命令, tfBinaryURL如何使用参考上面的说明

     
    ​x
    (tensorflow)$ pip install --ignore-installed --upgrade tfBinaryURL
    ​

  • 然后进行验证

验证安装

如果你是通过anaconda进行安装的, 那么你需要像之前一样激活你的tensorflow环境.

运行下列命令

 
xxxxxxxxxx
$ python
 
xxxxxxxxxx
# Python
import tensorflow as tf
hello = tf.constant('Hello, Tensorflow')
sess = tf.Session()
print(sess.run(hello))

如果你得到下面的结果, 说明安装成功!

 
xxxxxxxxxx
Hello, TensorFlow!

否则, 你可以看看tensorflow在github上面的issue和这些常见问题进行修正
